# Uniform Recommended Grading System

**Description:** A student may earn five letter grades on the basis of his/her performance in a course. The letter grades A, B, C, and D are considered passing grades. The grade F is the failing grade.

---

## Grading Scale

| Numerical Score | Letter Grade | Grade Point |
| :--- | :--- | :--- |
| 80% and above | A+ | 4.00 |
| 75% to less than 80% | A | 3.75 |
| 70% to less than 75% | A- | 3.50 |
| 65% to less than 70% | B+ | 3.25 |
| 60% to less than 65% | B | 3.00 |
| 55% to less than 60% | B- | 2.75 |
| 50% to less than 55% | C+ | 2.50 |
| 45% to less than 50% | C | 2.25 |
| 40% to less than 45% | D | 2.00 |
| Less than 40% | F | 0.00 |


---

## Special Grades & Policy

| Grade | Description | Grade Point | Policy Note |
| :--- | :--- | :--- | :--- |
| **F\*** | Failure | 0 | A 0 grade point in F Letter Grade is applied only for the calculation of the grade point average. |
| **U\*** | Unsatisfactory | 0 | Credits for courses with these grades do not apply towards graduation and are not used for the calculation of the grade point average. |
| **I\*\*** | Incomplete | 0 | Credits for courses with these grades do not apply towards graduation and are not used for the calculation of the grade point average. |
| **P\*\*\*** | Pass | 0 | Credits for courses with these grades are required for graduation but are not used for the calculation of the grade point average. |
| **R\*\*** | Repeat/Retake | 0 | Credits for courses with these grades do not apply towards graduation and are not used for the calculation of the grade point average. |
| **S\*\*\*** | Satisfactory | 0 | Credits for courses with these grades are required for graduation but are not used for the calculation of the grade point average. |
| **W\*\*** | Withdrawal | 0 | Credits for courses with these grades do not apply towards graduation and are not used for the calculation of the grade point average. |
